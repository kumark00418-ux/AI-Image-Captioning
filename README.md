# 🖼️ AI Image Captioning using Gen AI

## 📌 Project Overview
An AI-powered system that automatically generates 
captions for images using BLIP model.

## 🛠️ Technologies Used
- Python
- Hugging Face Transformers (BLIP Model)
- Gradio (UI)
- Google Colab

## 🚀 Live Demo
👉 https://huggingface.co/spaces/krishan17/ImageCaptioning

## ▶️ How to Run
1. Open notebook in Google Colab
2. Run all cells
3. Upload any image
4. Get AI generated caption!
